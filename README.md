Prerequisites:
JDK 17+
MySQL database instance
MYSQL Connector jar file

Steps to Run the Project:
1. Create .env.local file outside all folders in root directory

2. Create a variable called MY_SQL_PASSWORD in the .env file and enter your database password

3. Copy all of food_delivery_db.sql into a SQL file on your database instance

4. Run the copied SQL line by line on your database instance

5. Run LoginUI.java using JDK run icon (should be in the top right).

6. Create an account by using "Create Account" button and login using that account information OR use this account below:

User - tester3@gmail.com

After this, you should be logged in and able to use the app.
